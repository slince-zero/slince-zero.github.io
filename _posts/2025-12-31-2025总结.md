又到了做年度总结的时候，每年到这个时候似乎总是有着不一样的心境。去年这个时候，刚刚失业，心态比较焦虑，在写去年的年终总结的时候，感觉文字字里行间都是围绕着工作相关，但是今年准备写下这些文字的时候，似乎有些不一样了。

### 工作

今年年初，大概一月份的时候，入职了一家创业公司，整体给我的感觉就是一个字 —— 忙。我是在年前一周入职的，从入职以后，需求就没有断过，今年一整年似乎都是在忙碌中度过，有时候，周末也还在搞需求，当然，其实今天本来也是不例外的。为什么说 “本来”，因为我们公司要换新的办公室了，机房需要搬迁，运维大哥们这会估计还在机房里安装机器。

还是很感谢我现在的这家公司，虽然很忙，但是同事人都很好，大家也都很热情，在工作上，遇到不会的问题，也都会帮助我，指导我。这里不得不说，我的 mentor 和组里的另一个大哥，就到他 A 吧。我 mentor 是一个东北人，很热情，代码能力非常强，我遇到不会的时候去问他，通常是要带着上下文，不过可能我说一半，或者说一点，他就已经知道问题在哪了，有的时候他会直接告诉我解决方案，有的时候会给我关键字，让我自己去检索解决。另一个大哥 A，这位大哥可就太厉害了，真的很强，首先是他一直在坚持刷力扣，真的是每天都在刷，我也是从我 mentor 口中知道的，而且他的代码思维能力也非常强，尤其是在看代码的时候，我可能刚开始看一两行，他就已经理解了，并且能够迅速指出问题点。

我记得最近和 A 一起开发一个需求，我做前端，他开发 node 端，他很快就写完了，我这边进度有些慢，我在写一些函数拿数据的时候，他提了很多建议，比如对于一些 UI 重复渲染的东西，可以用递归，而且一些数据用递归去写，可以简化思考过程，这是我在他身上学到的东西，而且大哥现在还在读 MBA，真的很厉害，我看过他的 GitHub 更新记录，几乎可以说每天都在更新，反观我自己的 GitHub 这一年里，更新的非常不频繁，有几个月是直接断档的。

我记得有一次周会上面，A 分享过，他的乐趣就是写代码，如果换做其他人来说这句话，我可能保持怀疑态度，但是对于他，我却是深信不疑。

另外的话，自己也确确实实从他们身上学到了很多，比如一些 debug 技巧，让写代码写起来更舒服，也从后端大哥那里学了很多 docker 的用法，现在可以搭配大模型去写 dockerfile 了，一些基础的 docker 命令也是可以的了。

这一年里，在工作上还是有些点可以拿出来谈谈的，比如：

1. 在聊需求的时候，一定要保证自己有一定的上下文，比如完整看过需求文档或者跟同事了解这个需求有没有其他背景，这样再去跟产品和其他同事沟通的时候，能够大幅度提高沟通效率。

2. 不要不好意思，用我们老大的话来说，职场是一个比较严肃的场景，大家在一起共事，完成工作是前提。

3. 怀着学习的态度，认真对待每一个需求，沟通很重要，时间，人力成本，需求完成程度等等，都是可以聊的。

写到这里的时候，公司群里已经弹出，服务器已经装机完毕了，我试了下，已经可以了，运维大哥们真的太厉害了！


### 生活

生活上的场景，就比较平淡了，由于工作比较忙，基本每天到家之后，洗洗漱，刷会手机，基本就睡了，不过有段时间比较焦虑，经常熬夜。题外话，12 月，我已经加了一个月的班了，真的很累。

今年在不是特别忙的时候，见了几个朋友，朋友 A，还在读研究生，明年就要研究生毕业了，他目前在准备考公。朋友 B，也是明年研究生毕业，只不过女朋友考上了保定的一个有编制的岗位，大哥也是准备找保定那边的工作了。大家都在自己的节奏里，一步一步的走着，没有所谓的快慢，也没有所谓的对与错。

周末的时候，就去找女朋友，在他那边买菜做饭，偶尔还能用她的电脑打打游戏，另外就是我买的天幕在十月份的时候到了，六月初预定，历经四个月，也是不容易，不过还没有拼完，应该是只有十月份拼了一段时间，十一月份工作就非常忙了，导致周末也没有什么时间拼。

十一月份的时候，生了一场病，发高烧，最高的时候已经到了 39.3 度，那会直接就去急诊了，感觉下楼的时候，脑子整个已经乱成一锅粥了，走路都有点摇摇晃晃的，女朋友为了照顾我，他自己也被传染了，哎，后面就是我好了，她又病了，不过好在，她的病比较轻，基本一周就恢复了，不过就是一直在咳，已经吃了快一个月的中药了，医生说，有肺热，这里我也不是很懂，只是感觉咳的时间太长了。虽然后面也去医院检查过，但也并没有查出来什么，不过写下这些文字的时候，她已经不怎么咳了。

在这里感谢，女朋友的照顾，也感谢一直陪伴彼此的我们。



### 读书

今年买了一个 kindle，感觉应该是今年买的最好也是最实用的一个物品了，因为我每天坐地铁通勤，时间又比较久，有时候又不想看手机，最后还是入手了一个 kindle，每天坐地铁的时候都可以看一会，还是很棒的。

我看到第一个作品是 《龙珠》，哈哈哈，应该会有很多人像我一样，又拿来看书，又拿来看漫画吧，感觉即使过了这么多年，依然很喜欢看龙珠。

然后看了余华的 《第七天》，故事大概讲述了一个男人在死后失去了记忆，然后在这七天里，每天都能遇到不同的人，每个人都有着不同的故事，有一些讽刺意味。其中，我比较喜欢的是男主的父亲，男主是他父亲在铁轨上发现的，后来他父亲开始照顾男主长大，但那会他父亲的年纪也不是很大，那会还认识了一个姑娘，为了跟姑娘约会，把男主寄放到朋友家，他也犹豫过，甚至把男主送到孤儿院，但是最后他选择放弃了那个姑娘，找回了男主，再后来，男主渐渐长大了，有了自己的故事，而老父亲也一直没有结婚，直到最后，自己快要死的时候，也是偷偷地藏了起来，怕给男主增添负担。

其实我一直停推崇一件事，那就是珍惜身边的人，不管是谁，即使这段关系可能长久，也可能短暂。

最近，还看了一本小说，《最后一个死去的女孩》，这本书就是一个悬疑类的故事了，没有什么特别的观后感，不过故事挺有意思的，推荐一读。

另外，《带着癌症上班的年轻人》，这本书也是我偶然间发现的，时间有些久了，这个应该是年中的时候读的，故事比较短，正如书名，是几个年轻人得了癌症，却还坚持上班的故事，其中有的人比较幸运，有药去治疗，生存期还变化，有些人就 ...

和他们比起来，我们这些人真的是太幸运了。我今年虽然不是大病一场，但是这场发烧也是让我难受了三天，这三天里，基本什么都干不了，疫情过后，感觉身体的耐疼痛能力降低了，除了伴有剧烈的头痛，身体也是非常痛苦难受。我在想，如果有一天，自己真的因为自己作死，而生病或者受伤，因此耽误了一些重要的事情，这才是真正让人后悔的事情吧。

不管怎么说，各位，还是要注意身体啊，健康真的真的太重要了。


### 最后

整体写下来，感觉今年的年度总结略显仓促，可能也是比较累的缘故吧，本来，想着上周六就写完的，可能工作太忙了，几乎连续三周，每天都搞到凌晨，身体确实有点顶不住了。

那今年就到这里吧，祝愿我们身体健健康康，迈向新的一年。

明年继续加油！
